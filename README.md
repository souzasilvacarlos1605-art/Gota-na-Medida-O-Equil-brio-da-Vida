# Gota-na-Medida-O-Equil-brio-da-Vida
Simulador interativo que ensina o manejo correto da água. O usuário clica na tela para umedecer um solo seco e fazer a planta crescer. O objetivo é manter o equilíbrio hídrico: evitar que a planta morra de sede ou que o solo encharque, combatendo o desperdício de água e promovendo a agricultura sustentável.
